# ModuleCore
[![](https://jitpack.io/v/Leonw2229/ModuleCore.svg)](https://jitpack.io/#Leonw2229/ModuleCore)
一个项目基础架构的简单抽象

### 1、设置jitpack仓库
```
allprojects {
  repositories {
    maven { url 'https://jitpack.io' }
  }
}
```
### 2、添加仓库依赖
```
implementation 'com.github.Leonw2229.module-core:module_arch:v0.0.1'
```
